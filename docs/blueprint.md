# **App Name**: GrowArt AI

## Core Features:

- Artwork Analyzer: Analyze uploaded artwork (JPG/PNG) using AI to determine its theme, color, mood, and style.
- AI Caption Generator: Generate short, catchy Instagram captions optimized for engagement and virality using an AI tool; includes variations like funny, poetic, or minimalist.
- Trending Hashtag Generator: Fetch real-time or simulated trending hashtags related to the artwork's theme using APIs or current Instagram trends.
- Interface & Design: Clean, minimal, modern UI (pastel or gradient tones). Display uploaded artwork on the left and generated caption + hashtags on the right. Fully responsive for mobile and desktop.
- Search History Feature: Users can view their past searches with filters by date, caption type, and artwork style with the option to re-use or edit old captions.
- Copy to Clipboard Option: Provide a button to copy the generated caption and hashtags for manual Instagram posting.
- AI Insights & Smart Suggestions: Show best time to post, trending hashtags, and AI tips (e.g., 'Use fewer hashtags today') for better reach.
- Free Collaboration Tab: Allow users to connect and collaborate by adding a 'Free Collaboration on (then instagram logo)' tab that links to https://www.instagram.com/let_scollab?igsh=Z2hqbTh6cmd0aWJi.
- About Us Page: Display information on the company GrowArt AI with an overview of the website and its goal.
- Privacy Policy Page: Clearly outline how uploaded images are used, data collection practices, and user agreements.
- Contact Us Page: Supply feedback and collaboration suggestions on ways to improve the tool with access to a form.
- FAQ Page: Contain typical website question, such as questions surrounding data practices and sharing policies

## Style Guidelines:

- Primary color: Soft pastel purple (#D8B4FE) to evoke creativity and artistic expression.
- Background color: Light gray (#F5F5F5) for a clean and modern feel; it ensures the focus remains on the artwork and generated text.
- Accent color: Gradient from pink (#FF6B6B) to orange (#FF8E53) to mirror the Instagram color scheme; it will also visually highlight CTAs and important features.
- Headline font: 'Belleza' (sans-serif) for titles and emphasis; has a stylish, design-conscious vibe.
- Body font: 'Alegreya' (serif) for main text, ensuring easy reading in longer pieces of text.
- Use modern, minimalist icons that match the aesthetic theme.
- Split-screen layout with artwork on the left and generated content on the right; responsive design for mobile and desktop.
- Subtle transitions and animations when generating content and on hover states for interactive elements.
